# 在位编辑

在位编辑是一个小组件，正常情况下显示为普通的文字，点击的话可以进入编辑状态。
这避免了新开页面或弹出窗口进行编辑，用户体验也不错。

## 使用方法

```jsx
import InplaceInput from '~/comp/inplace-input';

// 修改回调
const onChange = val => {
  // ...
}

<InplaceInput variant='h6' sx={{ flex: 1 }} text={info.name || ''}
  fontSize='large' onConfirm={onChangeName}
/>
```
