# 确认套路

经常会遇到需要用户确认的情况，例如删除某个资源需要用户确认一下，之类的。

ReactGO 的确认套路用法如下:

```jsx
import { useConfirm } from 'material-ui-confirm';

const confirm = useConfirm();

await confirm({
  description: '确定要干某件事情吗？',
  confirmationText: '确定',
  confirmationButtonProps: { color: 'warning' },
  contentProps: { p: 8 },
});

doSomeThing();
```

注意 `confirm()` 返回一个 `promise`，你需要 await，不然就不确认直接执行后面的代码了。
